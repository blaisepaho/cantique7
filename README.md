# Cantique SDA — Hymnes de l'Église Adventiste du 7ème Jour

Application web des 654 cantiques de l'Église Adventiste du 7ème Jour.

## Accéder à l'application

**[https://blaisepaho.github.io/cantique7/](https://blaisepaho.github.io/cantique7/)**

## Fonctionnalités

- 654 cantiques complets avec texte intégral
- Recherche par titre ou par mots dans le texte
- Lecture MIDI avec accompagnement musical
- Favoris personnalisés
- 5 thèmes visuels (Lumière, Nuit, Aurore, Forêt, Océan)
- Choix de la police et de la taille du texte
- Application installable sur mobile (PWA)
- Fonctionne hors connexion après le premier chargement
