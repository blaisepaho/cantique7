# Cantique SDA — Hymnes et louanges

Application web de 654 cantiques — hymnes et louanges.

## Accéder à l'application

**[https://blaisepaho.github.io/cantique7/](https://blaisepaho.github.io/cantique7/)**

## Fonctionnalités

- 654 cantiques complets avec texte intégral
- Recherche par titre ou par mots dans le texte
- Lecture MIDI avec accompagnement musical
- Favoris personnalisés
- 5 thèmes visuels (Lumière, Nuit, Aurore, Forêt, Océan)
- Choix de la police et de la taille du texte
- Application installable sur mobile (PWA)
- Fonctionne hors connexion après le premier chargement

## Installer l'application sur votre téléphone

L'application peut être ajoutée à l'écran d'accueil pour y accéder comme une application native, sans passer par le navigateur.

### Android (Chrome)

1. Ouvrez le lien dans **Chrome**
2. Appuyez sur le menu **⋮** (trois points en haut à droite)
3. Sélectionnez **"Ajouter à l'écran d'accueil"**
4. Confirmez en appuyant sur **"Ajouter"**

L'icône apparaît sur votre écran d'accueil et l'application s'ouvre en plein écran, sans barre de navigation du navigateur.

### iPhone / iPad (Safari)

1. Ouvrez le lien dans **Safari**
2. Appuyez sur l'icône **Partager** (carré avec une flèche vers le haut, en bas de l'écran)
3. Faites défiler et sélectionnez **"Sur l'écran d'accueil"**
4. Appuyez sur **"Ajouter"**

L'icône apparaît sur votre écran d'accueil et l'application s'ouvre comme une application native.
